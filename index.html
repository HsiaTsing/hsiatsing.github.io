Selected Work

  1. 3D focal feature extraction

    2. We make an initial attempt to propose an learning based generic modeling approach for user-central definition and automatic extraction of features on arbitrary shapes.

  2. Shape Sequence Completion from Sparse Samples

    3. Given a few sparsely-sampled models along the temporal axis, our method produces a dense shape sequence which gives rise to natural transition from one key-frame to its succeeding one.

  3. Screen space SPH fluid rendering

    4. This is an implementation of the Screen Space Fluid Rendering for Games using OpenGL and GLSL.

  4. Details enhancement of SPH fluid

    5. This method is used to produce foams and sprays of SPH fluid (particle systems)

  5. Real-time X-Ray simulator

    6. This is a simple real-time X-Ray simulator program using CUDA for GPUs from NVidia. It takes triangulated surface models as input and produces their X-Ray images.

  6. Data-driven heartbeat simulation

    7. This is a part of our PCI surgery simulator, the heart-beating sequence is reconstructed from 4D MRI data.

  7. Haptics-equipped PCI simulator

    8. We advocate a haptics-equipped interactive prototype system towards Percutaneous Coronary Intervention (PCI) surgeons training and patient-specfic surgery rehearsing, which can afford trainees the opportunity to approximately experience the entire PCI procedures and customized emergency cases that might occur in common clinical settings.
